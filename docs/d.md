# Phase 4: EV機能とHUDの実装

仕上げとしてEV特有の機能とHUDを実装してください。

## 1. インホイルモーターとトルクベクタリング (TV) (Step 3)
後輪(RL, RR)の `F_drive` 計算ロジックを実装します。

### 基本駆動力
- `F_base = ThrottleInput * MaxDriveForce`

### 目標ヨーモーメント係数 (M_req)
- `M_req = SteerInput * Speed * TVGain`
- 直感的な挙動: 「右にハンドルを切ったら、左後輪を強く、右後輪を弱く」

### 左右配分
- **左後輪 (RL)**: `F_drive_RL = 0.5 * F_base - M_req`
- **右後輪 (RR)**: `F_drive_RR = 0.5 * F_base + M_req`

## 2. HUDコンポーネントの実装
以下の情報を表示するHUDを作成してください。
- **速度**: m/s -> km/h変換して表示。
- **各輪の垂直荷重 (Fz)**: バーグラフ等で可視化。
- **後輪駆動力 (Fx)**: バーグラフ等で可視化。
- **ステアリング角**: 数値またはインジケータ。

## 3. ビジュアル調整
- 車体のMeshは半透明にし、タイヤの位置やサスの沈み込みが見えるようにしてください。
- タイヤ（WheelVisual）の位置と回転を物理計算の結果に合わせて更新してください。
